### Component Annotations:
- **Role Sidebar**: Hierarchical role listing with user counts
- **Permission Categories**: Organized by functional areas
- **Bulk Actions**: Matrix view for efficient permission assignment
- **Role Templates**: Pre-defined permission sets for common roles
- **Audit Trail**: Track permission changes with timestamps

### Interactions:
- **Category Toggles**: Expand/collapse permission groups
- **Bulk Selection**: "Select All" for category-level permissions
- **Inheritance Indicators**: Show inherited vs. explicit permissions
- **Conflict Resolution**: Highlight and resolve permission conflicts
